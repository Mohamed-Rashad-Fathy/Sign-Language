#Steps to run the program

1-Download the code from github repository 

2-Open visual studio code or any another enviroment to run the code 

3-if you will using visual studio code Download python interpreter version 3.9.10 because mediapipe library work with some versions of python's interpreters 

4-Download mediapipe 0.8.11

5-Download OpenCV 4.5.3.56 

6-Download numpy  1.22.0

7-Download Tensorflow 2.5.0

8-Download scikit-learn 1.4.1.post1

9-Download matplotlib 3.5.3

10-Run the app and Move your hand in front of the camera, changing the position and shape of your fingers to see the expected text
